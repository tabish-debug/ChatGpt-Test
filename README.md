# ChatGPT-Test

This is assignment from Aegasis Labs in python (Flask) to build some functions using chatgpt API like

create prompt
get response from API
update prompt 
etc

#### MacOS SetUP 

```
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```

#### For Test

```
pytest
```
